# chromium-cache-dir-cleaner
帮助您扫描全盘查找Chromium的缓存文件夹并删除

现在，越来越多的Windows程序使用了Electron开发，这使得我们的电脑中存在了大量零散的Chromium缓存文件夹。<br>
本脚本旨在解放双手，省去您逐个应用查找和清理缓存文件夹的时间，直接运行脚本即可开始自动化扫描全盘并清理！
